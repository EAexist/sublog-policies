# 2. 取得する情報

## 2.1 アカウント情報

ユーザーが Google OAuth を使用してログインする場合、当方は以下の Google アカウント情報を取得する場合があります。

- メールアドレス
- 表示名

{{serviceName}} は、ユーザー名・パスワード方式による認証を提供していません。


## 2.2 Gmail メタデータ情報

{{serviceName}} は、ユーザーの明示的な同意および Google OAuth 認証プロセスを通じて、以下の Gmail API スコープを利用します。

- https://www.googleapis.com/auth/gmail.metadata

当該スコープにより、以下の限定的なメールメタデータを一時的に処理する場合があります。

- 件名
- スニペット
- 送信者情報
- メッセージ内部日時

{{serviceName}} は、メール本文全体へアクセスまたは閲覧することはありません。

Gmail メタデータは、サービス提供のためにメモリ上で一時的に処理され、データベースへ永続的に保存されることはありません。


## 2.3 派生データ

当方は、Gmail メタデータの処理によって生成された以下の分析結果を保存する場合があります。

- 抽出されたキーワード
- 分類結果
- サブスクリプション関連分析データ

これらのデータは、本サービスの提供および機能改善のために利用されます。


## 2.4 分析情報

当方は、サービス品質および安定性向上のため、第三者分析ツールを通じて限定的な利用情報および技術情報を収集する場合があります。

これには以下が含まれます。

- デバイスおよびブラウザ情報
- 一般的な利用統計
- パフォーマンスおよびエラーデータ

# 3. 情報の利用目的

当方は、以下の目的で情報を利用します。

- サブスクリプション分析機能の提供
- サブスクリプション関連メールの識別
- サブスクリプション関連キーワードの抽出および分析
- 分類精度およびサービス機能の改善
- 認証およびアカウント管理
- サービスパフォーマンスおよび安定性の監視
- 不正利用、詐欺、または不正アクセスの防止

当方は、個人情報または Gmail データを第三者へ販売しません。

また、Gmail データを広告目的で利用しません。

# 4. AI による処理

{{serviceName}} は、Google AI Studio を通じて提供される Google Gemini API を利用し、Gmail メタデータを処理します。

利用目的は以下の通りです。

- メール分類
- キーワード抽出
- サブスクリプション検出および分析

処理対象は、件名およびスニペットなどの限定的な Gmail メタデータに限られます。

AI 処理は、本サービスの提供および改善目的のみに使用されます。

# 7. 第三者サービス

本サービスは、以下の第三者サービスを利用する場合があります。

各第三者サービスは、それぞれ独自のプライバシーポリシーおよび利用規約に基づいて情報を処理する場合があります。

# 8. データ保持期間

当方は、本サービス提供および正当な運営目的に必要な期間に限り情報を保持します。

- Gmail メタデータはメモリ上で一時的に処理され、永続保存されません
- アカウント情報および派生分析データは、アカウントが有効な期間保持される場合があります
- 一部の運用ログおよびバックアップデータは、最大 {{DATA_RETENTION_PERIOD}} 保持される場合があります
- ユーザーは {{contactEmail}} へ連絡することで、アカウントおよび関連データの削除を要求できます

法令遵守、セキュリティ、または不正防止目的のため、一部情報を追加保持する場合があります。

---

# 9. データセキュリティ

当方は、情報の不正アクセス、漏えい、悪用、または紛失を防止するため、合理的な技術的・組織的安全管理措置を実施します。

ユーザー関連情報へのアクセスは、保守、サポート、セキュリティ、または法令遵守上必要な場合に限り、権限を有する者に限定されます。

ただし、インターネット通信および電子保存方法に完全な安全性を保証することはできません。

---

# 10. 国際的なデータ移転

本サービスは {{operatorLocation}} から運営されています。

利用するインフラまたは第三者サービス提供者によっては、情報がユーザーの居住国外で処理または保存される場合があります。

これらの国や地域では、ユーザーの居住国とは異なるデータ保護法が適用される場合があります。

ユーザーは、本サービスを利用することにより、サービス運営上必要な範囲で国際的なデータ移転が行われることを理解し、同意するものとします。

違反が判明した場合、当方は当該情報を削除するための合理的な措置を講じます。
